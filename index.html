SMTP OVH, LWS, Infomaniak

Dans Réglages > Provider d'envoi, choisissez votre hébergeur. Les paramètres SMTP se pré-remplissent.

  • OVH : ssl0.ovh.net port 465
  • LWS : mail.lws.fr port 465
  • Infomaniak : mail.infomaniak.com port 587
  • IONOS : smtp.ionos.fr port 587
  • MailChannels : gratuit CF Workers — SPF include:relay.mailchannels.net
La limite "480" = votre dailyLimit dans la campagne (Automatisation > Par jour).
Cloudflare Worker
  • Workers > Créer > coller worker.js
  • KV Binding : PGI_KV
  • Cron : */5 * * * *
  • Secrets : GMAIL_CLIENT_ID, GMAIL_CLIENT_SECRET, GMAIL_REFRESH_TOKEN
  • SMTP : SMTP_HOST, SMTP_PORT, SMTP_USER, SMTP_PASS
Quotas
  • Gmail : 500/j — Workspace : 2000/j
  • Outlook : 300/j — M365 : 10000/j
  • OVH : ~200/j — LWS : ~500/j
  • SendGrid / Mailgun : 100000/j
  • MailChannels : illimité

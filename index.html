⚡ Dashboard

Le Dashboard est votre centre de contrôle en temps réel. Il affiche :

  • KPIs globaux : emails envoyés, taux d'ouverture, non ouverts, clics liens, désabonnés, SMS envoyés
  • Filtre par période : Aujourd'hui, Hier, 7 jours, 30 jours, Tout, ou dates personnalisées
  • Historique email : chaque email envoyé avec son statut (envoyé / ouvert / erreur / désabonné)
  • Historique SMS : chaque SMS avec statut lu / non lu / STOP
  • Campagnes actives : avancement en temps réel (sync toutes les 30s), cadre Clics Liens par campagne
💡 Les données sont synchronisées avec le Worker Cloudflare quand il est configuré — accessible depuis n'importe quel appareil dans le monde.
📧 Campagnes Email — v38

Créez des campagnes d'emailing avec automatisation complète :

  • Section 1 — Destinataires : Copier-coller des milliers d'adresses d'un coup, importer CSV/Excel, ou charger depuis vos listes
  • Section 2 — Template : Choisissez un template sauvegardé ou rédigez directement dans l'éditeur intégré
  • Section 3 — Automatisation : Cadence multi-unité (seconde/minute/heure/jour), limite journalière stricte, durée, ordre d'envoi
  • Section 4 — Planification horaire : Jours de la semaine + créneaux horaires précis par jour
  • Section 5 — Anti-bannissement : Délais aléatoires entre envois, quotas par provider
💡 Pause & Reprise intelligente (v34) : quand vous mettez en pause, le bouton devient ▶ Reprendre. Cliquer reprend exactement au contact suivant — le compteur journalier et la durée ne sont PAS remis à zéro. Vous ne perdez aucun envoi.
💡 Multi-appareils : si vous pausez depuis votre téléphone, le PC se synchronise automatiquement et affiche aussi le bouton ▶ Reprendre avec le bon état.
🤖 Automatisation & Cadence — v34

La cadence d'envoi se configure sur 4 unités indépendantes, avec une case à cocher pour activer/désactiver chacune :

  • Par seconde ☑ : ex. 1/s = max 1 email toutes les secondes (attention : plan gratuit Gmail = ~1/s max)
  • Par minute ☑ : ex. 10/min = 10 emails envoyés par minute
  • Par heure ☑ : ex. 300/h = max 300 emails par heure
  • Par jour ☑ : ex. 300/j = plafond absolu — le Worker s'arrête exactement à 300, jamais un de plus

Le Worker applique la contrainte la plus restrictive. La cadence effective s'affiche en temps réel dans le résumé.

⚠️ Limite journalière = loi : si vous réglez 300/j, le Worker envoie 300 et s'arrête. Le lendemain, le compteur repart à zéro automatiquement.
💡 Durée : de 1h à ∞ (infini = tourne jusqu'à arrêt manuel ou fin de liste)
🕐 Heures & Jours d'envoi

Dans la section Planification horaire de chaque campagne :

  • Choisissez ⚡ Continu 24h/24 ou 📅 Créneaux horaires
  • En mode créneaux : cochez les jours actifs (Lun à Dim) et définissez les heures De/À pour chaque jour
  • Le Worker n'envoie qu'aux créneaux définis, même téléphone éteint
💡 Exemple noctune : Lun–Dim · 23h00→06h00 + 300/j = envoi automatique la nuit, 300 maximum, sans intervention
📊 Quotas & Multi-campagnes

Vous pouvez lancer plusieurs campagnes simultanément. Le Dashboard affiche en temps réel :

  • Le quota consommé par campagne (envoyés / limite du jour)
  • La position courante dans la liste de contacts (ex: 47/300)
  • Une alerte rouge si la somme des limites dépasse le quota provider
⚠️ Limites officielles : Gmail 500/j (perso), Gmail Workspace 2000/j, Outlook 300/j. Ne dépassez pas ces seuils sous peine de suspension.
🚀 Envoi Rapide

Pour envoyer un message ponctuel et urgent sans créer de campagne :

  • Rédigez votre email ou SMS directement dans l'éditeur
  • Collez les adresses ou numéros destinataires (des milliers d'un coup)
  • Cliquez "Envoyer maintenant" — envoi immédiat via le Worker Cloudflare
  • Chaque envoi est tracé dans le Dashboard et les Analytics
💡 L'éditeur de l'Envoi Rapide supporte les mêmes fonctionnalités que l'éditeur Templates : images, liens, vidéos, tableaux, variables.
✏️ Templates & Éditeur
  • Éditeur Visuel : mise en forme complète (police, taille, couleurs, alignement, listes)
  • Images : insérez par URL ou uploadez — les images sont hébergées sur votre Worker Cloudflare (90 jours)
  • Liens cliquables : sur texte, image ou numéro de téléphone (tap-to-call)
  • Code HTML : collez votre HTML avec aperçu en direct
  • Auto-sauvegarde : toutes les 30 secondes + Ctrl+S
⚠️ Templates = stockage local : les templates sont sauvegardés dans le navigateur (localStorage). Pour les transférer entre appareils : Réglages → Exporter JSON, puis importez sur l'autre appareil.

⚡ Cloudflare Worker v5.3 — Le moteur 24h/24

Le Worker est indispensable pour l'automatisation. Il tourne 24h/24 même appareil éteint :

  1. Créez un compte sur cloudflare.com
  2. Workers & Pages → Créer → Créer un Worker → collez le code Worker v5.0
  3. Paramètres → Variables → KV Namespace Bindings → Ajoutez PGI_KV
  4. Triggers → Cron Triggers → Ajoutez */5 * * * * (toutes les 5 min)
  5. Secrets requis : GMAIL_CLIENT_ID, GMAIL_CLIENT_SECRET, GMAIL_REFRESH_TOKEN, GMAIL_SENDER
  6. Copiez l'URL du Worker dans Réglages → Cloudflare Worker
💡 Le Worker stocke tout dans Cloudflare KV : campagnes, contacts envoyés, analytics, statuts. La synchronisation entre appareils est automatique.
🌍 Synchronisation multi-appareils (v34)

Hébergez votre fichier HTML sur GitHub Pages (gratuit) pour y accéder de n'importe quel appareil :

  1. Créez un repo GitHub et uploadez le fichier comme index.html
  2. Settings → Pages → Source : main → Save
  3. Votre URL : https://votrecompte.github.io/nom-du-repo
  4. Ouvrez cette URL sur téléphone, tablette, PC — tout est synchronisé

Ce qui se synchronise automatiquement (via Worker KV) :

  • ✅ Campagnes actives — statut, contacts envoyés, position courante, quota du jour
  • ✅ Pause/Reprise — pausez sur téléphone, reprenez sur PC au même contact
  • ✅ Analytics — tous les envois, ouvertures, désabonnements
  • ⚠️ Templates — stockage local uniquement (exportez JSON pour transférer)
💡 La sync est automatique toutes les 30 secondes. Cliquez 🔄 Sync dans le Dashboard pour une sync immédiate.
⚙️ Réglages
  • Compte email : Gmail (OAuth GIS), Outlook (OAuth), ou adresse personnalisée
  • Adresse par campagne : chaque campagne peut utiliser un compte d'envoi différent
  • Worker Cloudflare : URL + test de connexion en un clic
  • SMS : Twilio, Vonage, ou webhook personnalisé (tout provider compatible)
  • Export JSON : sauvegarde complète de toutes vos données (contacts, campagnes, templates, analytics)
📈 Tracking & Analytics
  • Dashboard : taux d'ouverture email, SMS lus, désabonnés, erreurs, position courante par campagne
  • Historique filtrable : par jour, semaine, mois ou période personnalisée
  • Pixel de tracking : le Worker insère un pixel invisible dans chaque email pour détecter l'ouverture
  • Google Analytics 4 : chaque action trackée (optionnel — remplacez G-XXXXXXXXXX dans le code)
